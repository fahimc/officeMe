# officeMe

## Modular Office Grid

The Codex-ready modular 2D office app is in [`modular-office-grid/`](./modular-office-grid/).

```bash
git clone https://github.com/fahimc/officeMe.git
cd officeMe/modular-office-grid
python -m http.server 8080
```

Open `http://localhost:8080`.

Codex instructions are provided in [`modular-office-grid/AGENTS.md`](./modular-office-grid/AGENTS.md).
